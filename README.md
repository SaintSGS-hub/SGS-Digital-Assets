Ele funciona como a biblioteca estática da marca.

## Visão

O SGS Digital Assets nasce como a camada de suporte visual da SGS Digital, reunindo os arquivos que sustentam a atmosfera premium, escura, seletiva e autoral do site principal.

Este repositório não é apenas um depósito de imagens.  
Ele é a base estética da presença digital da SGS.

Aqui ficam armazenados os elementos responsáveis por manter consistência visual, identidade, performance e reaproveitamento entre páginas, experiências e futuras extensões da marca.

O projeto foi organizado para servir:

- imagens institucionais;
- banners de apresentação;
- assets de hover e cards visuais;
- favicons e manifest;
- cursores personalizados;
- fonte customizada Yayusa;
- música ambiente;
- modelos e componentes 3D;
- arquivos de apoio usados no deploy principal;
- recursos estáticos consumidos por HTML, CSS e JavaScript externos.

A função principal deste repositório é permitir que o site da SGS Digital carregue seus elementos visuais de forma centralizada, limpa e escalável.

## Relação com a SGS Digital

Este repositório serve diretamente ao projeto principal:
